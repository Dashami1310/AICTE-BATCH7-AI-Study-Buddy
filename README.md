# AI Study Buddy

AI Study Buddy is an intelligent learning assistant that helps students understand academic topics, summarize study notes, and generate practice quizzes. The application provides an interactive interface where users can enter topics or notes and receive clear explanations, concise summaries, and self-assessment questions.

## Features

- Topic Explanation  
  Provides simple and clear explanations of academic concepts.

- Notes Summarization  
  Converts lengthy study material into concise bullet-point summaries.

- Quiz Generation  
  Creates multiple-choice questions for self-assessment and revision.

## How It Works

Users enter a topic or paste study notes into the application.  
The system processes the input and generates educational content such as explanations, summaries, or quizzes, which are displayed instantly in the interface.

## Usage

1. Open the application  
2. Enter a topic or paste notes  
3. Select the desired feature  
4. View generated explanation, summary, or quiz  


## Project Purpose

This project demonstrates how intelligent learning tools can support students in understanding concepts, reducing study time, and improving revision through automated educational assistance.
